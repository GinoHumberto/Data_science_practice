\begin{enumerate}
    \item Definir el problema ¿Qué queremos resolver?
    \item Explorar los datos ¿Quiénes forman parte y sus características?
    \item Limpieza y preparación.
    \item Análisis y modelado.
    \item Evaluación.
\end{enumerate}

\section{Objetivo}

Constuir una herramienta que al darle los datos de un pasajero determine si sobrevive o no.

\section{Criterios}

Queremos determinar si sobrevivio o no, por lo tanto estamos frente a un problema de clasificación. Para resolverlo tenemos que basarnos en quienes sobrevivieron y quienes no.

\subsection{Preguntas}

\begin{itemize}
    \item ¿Era de primera clase?
    \item ¿Qué edad tenía?
    \item ¿Tenia dinero para pagar un camarote cerca de la cubierta?
\end{itemize}

\begin{nota}
    El problema de define como usar las pistas del pasado para predecir un resultado binario (si/no).
\end{nota}
